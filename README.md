## 经验
每个项目都好好想想。。。你想传达给  面试官什么。。。要是面试官没问，你就找机会自己说出来.我阿里一面的经验就是，，，如果面试官问你的东西，你很懂。那就要回答到自己满意，不要后悔。尽量把你懂得都传达给面试官
二面，就是 竟然还有智力题。。。  面试官的问题，你不会没关系不慌。。。不急着给面试官答案，的。 面试官不是要看你会不会，而是看你分析问题的能力。。。
